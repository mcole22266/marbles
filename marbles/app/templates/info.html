Site Info

This website in built primarily with Flask - a Python micro web-framework mainly due to it's modularity. One of the goals of this website was to build it entirely with a container-based architecture. Due to this goal, the Flask app sits inside a docker container which allows it to be isolated from any other services. Alongside the app container, there is a Postgres container that acts as the backend for website.

Container orchestration is handled by docker-compose which allows the entire application to be easily built on any machine. This aids greatly in deployment to production as the entire site can be up and running with a simple docker command.

As of this writing, the site is not yet deployed so further information regarding site info will be added upon completion of deployment. However, at this point, the plans are to host the site in an AWS EC2 instance. Currently, there already exists a CloudFront Distribution which points to an S3 Bucket hosting a static "Coming Soon" site. When this app is deployed on the EC2 instance, the CloudFront distro will simply be pointed to the new instance.
